扫盲桌面操作系统

今天我们主要讲讲生活中常见的操作系统，希望能让你了解一些除Windows以外的操作系统，并且能做出自己的选择 ：）
本文所有操作系统特指桌面端

P1 Windows

从安全性的角度上说，本文【不推荐】此系统，甚至不想写。但是由于windows系统比较广泛，所以简单写一些。
我认为，Windows系统【不推荐】的原因主要有如下几点

糟糕的开发团队

事实上我对系统开发并不了解，但是根据一些现有消息和个人的体验，还是有一些内容可说
1.Windows开发团队近几年被削减的很严重，微软已经把营销的重点放在了搞钱上，系统的开发已经非常混乱。
2.由多个团队开发，彼此之间根本没有协调性
3.祖传代码，架构老，每次都是在原基础上修修补补，bug自然是不用说

树大招风+自身漏洞

1.用户量多，所以定制的病毒自然多，毕竟是重点攻击对象
2.没有像linux一样完善的文件权限，比如我就干过把exe伪装成图片的事……
3.臭名昭著的ie以及很多漏洞

一些妥协的建议

叨叨半天，我知道一般人也离不开Windows，毕竟大部分游戏都需要Windows平台的支持（虽然一些Linux发行版也在尝试，不过效果嘛……），所以**建议你自己定制一下自己的系统，比如用个ltsc，定制一些预装应用 ——诸如此类，建议自己上网想办法

P2 Linux

这个估计大家听得多，用得少，也是本文介绍的重点

Linux的优点如下

1.发行版本多，个性化的可能性更高，可以因人而异，同时也避免了树大招风
2.开源，安全性有保障
3.玩法多样（只要肯折腾）
4.广泛的社区——基本上可以做到有求必应
5.完善的软件包
6.适配性不错很多人说Linux兼容性不好，其实本人认为是够用的，archlinux针对中文用户就有很好的支持，比如
archlinuxcn中文本地化的文档
archlinux丰富的软件库(链接里是一个第三方源，在国内速度很快)包含wine-wechat等各种应用
另外就算你需要的应用找不到，你也可以运行一个虚拟机（小提示：linux很多桌面版本功耗很低，同配置机型开虚拟机比Windows或macOS流畅很多）

推荐几个linux发行版

archlinux（可以原版，也可以manjaro等等，安装稍有难度，不过有图形化安装界面）archlinux的个性化程度比较强，有点像毛坯房，初次上手有难度

ubuntu （在Linux中人数较多，不过有些商业化，出现过一些丑闻）不过好处是几乎可以拎包入住

Debian 稳定经典，本人用得少，所以不过多评论
